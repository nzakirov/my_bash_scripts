# Translate

Скрипт позволяет работая на десктопных Linux системах выделив текст в абсолютно любом окне (терминал, браузер, документ pdf и тд) перевести его на русский с любого языка, используя возможности утилиты переводчика.

Зависимости,установленные в системе:  
- **`trans`** - консольный переводчик
- **`xsel`** (Программа командной строки для получения и настройки содержимого выделения в графической среде)
- **`zenity`** ( Отображение графических диалоговых окон из сценариев оболочки) или подобная (зависит от дистрибутива)

Алгоритм работы заключается в следующем. При выделении мышкой текста, при помощи **_xsel_** он извлекается в переменную, которая передается в **_translate_**  и выводится при помощи утилиты отображения диалогового окна.

Скрипт нужно сделать исполняемым и поместить в PATH пользователя, назначив какую либо горячую клавишу, например Alt + 5
Выделяем текст мышкой, нажимаем  горячую клавишу, получаем перевод в диалоговом окне.
